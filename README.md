# Harry Potter App
## Features
- Hogwarts Theme Login Page with House, Wand, Patronus, etc. as inputs...
- Main Page with Characters, Spells and Creatures displayed in Horizontal Recycler views, with a detailed description on click.
- Detailed Page of each(Characters, Spells and Creatures) with a feature of searching, filtering and sorting with different fields like House, Ancestry, role, etc.
- User can save their favorite characters, spells and creatures.
- A dedicated page is there where character, spells and creatures are in different tabs, showing the favorite ones, user can unfavorite their character from main page as well as from favorite page also.
- A dedicated Analytics page is there, where user can see the number of API fetched till now, number of successful API fetched and the number of failed ones, along with that Average time to call an API is also there.
- A Profile Page showing the details of the user. Along with number of favorite characters, creatures and spells.
- In the Profile Page, an edit button is there beside the profile image where user can select their favorite character and set it as their profile image.
- At the end a logout button is also there in the profile page, if the user wants to logout from the app.
